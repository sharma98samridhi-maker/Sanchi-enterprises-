# Sanchi Enterprises

## Business Website Project Documentation

### Overview
This project aims to create a responsive and user-friendly website for Sanchi Enterprises to showcase products and services.

### Features
- Home Page
- Product Listing
- Service Details
- Contact Form
- User Authentication

### Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- MongoDB

### Installation
1. Clone the repository: `git clone https://github.com/sharma98samridhi-maker/Sanchi-enterprises-.git`
2. Navigate to the project directory: `cd Sanchi-enterprises-`
3. Install dependencies: `npm install`
4. Run the application: `npm start`
